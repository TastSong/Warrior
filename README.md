## 2020.1.31 上传原始项目

此项目是基于[Taikr](https://www.taikr.com/) 《武士2》的项目的二次开发，开发的目的主要是实现游戏的安卓化。

主要有一下目标：

- [x] 加入`AndroidSDK`,发布安卓端游戏
- [x] 实现基本游戏控制，例如行走、发射技能
- [ ] 给游戏人物加血条
- [ ] 添加游戏加载页面
- [ ] 给游戏添加推出按钮
